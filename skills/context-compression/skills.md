# Context Compression Skill

## Description

Implements conversation history compression algorithms and context management for efficient handoffs.

## Capabilities

- Design compression strategies (summary, sliding window, hybrid)
- Implement token counting and limits
- Extract key facts and entities
- Preserve conversation state
- Optimize compression ratios
- Handle multi-turn conversations
- Maintain context coherence

## Triggers

- When implementing compression algorithms
- When optimizing context size
- When handling large conversation histories
- When token limits are exceeded
- When context quality degrades

## Handoff Conditions

- Needs LLM integration expertise
- Performance optimization required
- Edge cases in compression logic
- Complex summarization requirements
- Multi-language context handling

## Dependencies

- typescript-architecture (for type definitions)
- compression-algorithms (for advanced algorithms)

## Outputs

- Compressed context objects
- Key fact extractions
- Conversation summaries
- Token count reports
- Compression metrics

## Quality Standards

- Maintain >80% information retention
- Keep compression ratio under control
- Preserve critical context
- Handle edge cases gracefully
- Provide compression metadata
